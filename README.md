# Namma Metro Blog

**Namma Metro Blog** is a frontend project designed to showcase information and updates about the Namma Metro. This project is built using HTML, CSS, and Bootstrap to ensure a responsive and modern user interface.

## Features

- **Responsive Design**: The project is fully responsive, making it accessible across various devices.
- **Clean UI**: Utilizes Bootstrap components to provide a clean and professional look.
- **Easy Navigation**: Includes a simple and intuitive navigation bar for easy access to different sections of the blog.
- **Custom Styling**: CSS is used to enhance the default Bootstrap styles and add unique touches to the blog.

## Technologies Used

- **HTML**: Structure and content of the web pages.
- **CSS**: Styling and layout of the web pages.
- **Bootstrap**: Responsive design and prebuilt components.
